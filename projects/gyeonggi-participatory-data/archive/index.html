과제 개요

농촌공간 재구조화법 이행을 지원하기 위해, 현장전문가(활동가) 교육·현장조사와 공공데이터를 결합하여 표준화·재현가능한 데이터 구축 모델을 만든다. 일상생활권·자원·토지이용 정보를 데이터 사전·품질관리와 함께 정립하고, 시범지 여주시에서 실증한 뒤 경기도 15개 시군으로 확산하는 것을 목표로 한다.

  • 접근: 공공데이터 1차 매핑 → 시청 행정데이터 → 현장조사의 3단계 데이터 확보 전략
  • 공간단위: 법정리/행정리 단위(여주 234~235개), GPS 기반 현장조사
  • 확산 로드맵: 2026 여주 → 2027 안성 → 5년 내 경기 15개 시군

회의 내용

  • 2026-05-26 · 내부 회의 — 3차 교육-현장조사-워크숍 프로세스 설계, 활동가 15명·보상(1인 90만 원, 3회 분할) 조율. 프로젝트 방향을 ‘지표’ 수집에서 시행계획용 로우 데이터 수집으로 전환.
  • 2026-05-28 · 여주시 착수 미팅 — 여주 시범지 선정 배경(시행계획 단계 유일), 181→81개 후보지표 정리(공공 64% / 현장 36%), 항목별 데이터 확보처(빈집→건축과, 친환경→농관원, 외국인→법무부 등). 행정데이터 우선 확보 후 현장조사 보완 방향 확정.
  • 2026-06-08 · 내부 회의 — 181개 지표 분류(포털 127·공문 27·현장 20·중복 13), 용역사에 ‘지표’보다 ‘데이터 단위’로 질의하는 방식 제안, 기존 시군 기본계획 분석으로 ‘플러스 알파’ 항목 도출.
  • 2026-06-10 · 착수보고회(전문가 자문회의) — 181개 전수조사 비현실성 → 현장 필수 항목(빈집·유휴농지·위해시설) 집중. 공공데이터 ‘오류 찾기’ 방식의 주민참여 전략, 민감정보(축산·빈집) 공개/비공개 구분. Action items: 6/24 수집범주 정의, 6/26 좌표 표준양식, 6/28 우선순위 근거서 템플릿, 7/1 검증 프로세스, 7/15 빈집 교차검증.
  • 2026-06-11 · 내부 회의 — 공공데이터→시청→현장 3단계 전략 확정, 행정리 경계 SHP 부재 문제(대안: 마을회관 점·법정리 단위), 여주 234개 행정리 리스트업.

현재 진행상황

진척도 약 28% — 착수보고 완료, 공공데이터 1차 매핑·데이터 설계 단계.

  • ✅ 과업지시서·계약, 기술제안서 제출 완료. 여주 시범지 선정.
  • ✅ 착수보고회(6/10) 완료 — 용역사 포함 전문가 자문회의 형식.
  • ✅ 공공데이터 매핑 명세서 v1 — 38개 데이터셋 레지스트리, 시청 요청 11개 항목 압축.
  • ✅ 여주 인터랙티브 3D 지도 v1 — 법정리 123개 + 7종 실데이터 레이어(축산농가·경로당·공장·보호수 등). 경로당 데이터로 행정리 235개 초안 복원.
  • ✅ 여주 읍면별 마을 이력 딥리서치.
  • ⏳ 현장조사 준비 중(활동가 교육·조사 7~8월 예정).
