최근 회의

한국형 3-30-300 지표 기준 확정 회의 2026.05.13 · 33일 전

✓ 이번 회의 결정 사항

  • 3-30-300 기준 확정 — 층수별 버퍼(저층 30m/중층 55m/고층 100m), '30' 사적녹지 포함, '300' 300m 네트워크
  • 거점도시 7곳 대상지 선정, 분석단위를 행정동 → 집계구로 세분화
  • DeepForest 옥상 초록색 오탐 확인 → 산림청 가로수 데이터로 파인튜닝하기로

▶ 다음 회의·진행 업무

  • 산림청 가로수 데이터 수령(외장하드) 후 수관탐지 모델 파인튜닝 착수
  • 데이터 도착 전 거점도시 1~2곳 K-TES 점수 시뮬레이션 선행
이전 회의 3건 전체 보기 →

향후 주요 과제

  • 산림청 가로수 데이터 수령 → DeepForest 수관탐지 파인튜닝 (진행 중)
  • Sentinel-2 NDVI 파이프라인 구축 (5~10월 합성·구름처리·마스킹)
  • 거점도시 1~2곳 K-TES 점수 시뮬레이션 (데이터 도착 전 선행)
  • 전국(Tier1)·거점도시(Tier2) 3-30-300 지표 산정 및 K-TES 지도화

현재 진행 단계 · 팀 목적

현재 단계

'가. 실태조사'(9개국 비교) 완료, '나. 지표개발' 진입 단계. 산림청 가로수 데이터 수령 후 수관탐지 모델(DeepForest) 파인튜닝 준비 중. (진척 ~28%)

우리 팀의 목적

면적 중심 도시숲 정책을 형평성 기반 단계적 조성 전략으로 전환하고, K-TES로 우선 조성 대상지를 도출해 2026-11-30 최종 산출물로 제출한다.

향후 주요 일정 (D-day)

  • 2026.10 심포지엄 개최 (9~10월경) 마일스톤 D-108
  • 2026.11.30 최종 산출물 납품 마감 D-168
  • 2026.12.01 과업 공식 종료 마감 D-169
