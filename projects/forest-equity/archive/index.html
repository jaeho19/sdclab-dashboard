지금까지의 회의 기록과 전체 일정 (히스토리)

회의 기록 (9건)

  1. 산림청 미팅 안건 확정 — '다. 전략마련' 결과물 범위 상정안 2026.07.09

    ✓ 결정 사항

    • '다. 전략마련' 결과물 범위를 옵션 A(표준 가이드라인형)·B(사례 청사진형)·C(가이드라인+대표 청사진 3건 이원 산출) 중 택일로 발주처에 상정하기로 하고, 연구진 권고안은 옵션 C로 확정
    • 핵심 제안 — S2(구조토양·수트셀·연속 트렌치 등 지하 개량)만이 산림청이 단독으로 표준화·예산 배분할 수 있는 수단이며, 매뉴얼의 '신공법 도입 검토' 의무 조항의 빈칸을 본 과제가 채운다는 논리로 제시
    • 산림청 매뉴얼이 식재기반을 면적으로만 규정하고 토양체적(부피) 규정이 없음을 원문 확인 — 구도심 가로수 생육 부진의 구조적 원인 후보로 지목, 토양체적 규정 신설을 매뉴얼 개정 질의로 상정
    • 근거 미확인 재인용 수치 2건(도로 다이어트 매출 +5%, 가로수 매출 +9~12%)은 불사용 결정, 심층 리서치 2건에서 교차검증된 수치(식재대 최소폭 1.22m·대교목 토양체적 30㎥/주·도로 다이어트 임계 일 교통량 2만 대 등)만 채택
    • 식재간격은 일괄 8m 대신 수형급별 이원 산출(대교목 8m/소교목 4.5~6m)로 재계산하기로 — 5월 시뮬레이션 그루수는 전봇대·매설물 미차감 최대치로 사업량 약속 불가를 명시

    ▶ 다음 업무

    • 산림청 미팅에서 질의 Q1~Q8 확인 — 옵션 택일·청사진 3건 압축 동의, '이미지화' 수준(투시도/단면 상세/실시설계), 매뉴얼 개정 연계, A1·A2(도로 다이어트·지중화) 포함 여부, 가로수 위치 데이터 지자체 협조 공문, S2 시범사업 후속과제
    • 옵션 확정 즉시 착수 — 표준 파라미터 표·법제 정합 검토·S2 표준상세도(A/C), 대표 청사진 3건 제작(B/C), 수형급별 그루수 보정·공법 단가 리서치(공통)
  2. 3·30·300 지표 세부 설계·유형화 로드맵 회의 2026.07.01

    ✓ 결정 사항

    • 300 지표 접근성은 2단계로 산정 — 1안 공원 출입구 기준 도로 네트워크 300m 도달영역을 우선 시도하고, 결과가 미흡하면 2안 300m 원형 버퍼로 전환
    • 3·30·300 세 지표 모두 0~9점 점수화를 기본으로 하고, 필요 시 이분법(있음/없음) 파생 지도를 추가 제공
    • 건물 버퍼는 행정구 경계 밖 면적을 분모에서 제외해 정규화하고, 30 지표의 중첩 수관 폴리곤은 병합(Union) 후 면적 산출
    • '3' 지표 층수별 차등 버퍼 저층 0~30m·중층 30~55m·고층 55~100m 적용(13층 초과=초고층), 중층 55m 근거는 추가 보강
    • Tree Detection 지역별 정확도 편차 대응으로 외부 전문가 기술 자문을 받기로(7/10 금 10시), 토지피복도 대신 임상도로 산림 마스킹 검토
    • 가로수는 탐지 모델 대신 지자체 협조 공문으로 기존 데이터를 직접 확보, 유형화·투자 로드맵은 대전 산림청 미팅에서 발주처 요구 확인 후 보강

    ▶ 다음 업무

    • 대전(산림청) 미팅 일정 사전 조율 및 발주처 요구사항 리스트 요청 (~7/5)
    • Tree Detection 전문가 미팅(7/10) 질문 준비 및 임상도·건물 데이터 정제
    • 동대문구 대상 공원 출입구 300m 네트워크 접근성 1차 분석, 5/10/15분 구간 시험분석으로 점수 스케일 확정 (~7/15)
    • 층수 분류·경계 밖 버퍼·8㎡ 수관 가정·55m 버퍼 등 산식·가정의 선행연구 근거 문서화 (~7/19)
  3. 주간 회의 (모델 정확도·KEI 형평성 지도 통합) 2026.06.23

    ✓ 결정 사항

    • 동대문구를 우선 대상지로 탐지 → SAM 정답지 → 3m/30m/300m 분석까지 한 사이클을 완주해 워크플로를 먼저 검증하기로 결정
    • 수목 탐지 정확도는 LocateAnything=F1, SAM=mIoU로 분리 평가, 정량 평가는 동대문구 육안 품질 검토 이후로 미룸
    • '뱀 현상'(객체 과분할·오연결, 크롭 약 120장당 1장 빈도) 오류 이미지는 선별 재처리 후 분석
    • KEI 기온 지표 정규화 문제(시·군 단위 제공 → 자치구 내 분모 0) 대응으로 LST 등 대체 소스 탐색, 동대문구 우선 격자 단위 점수화 검토
    • 분석 단위 이원화 — 우선순위는 행정동, 변화상 제시는 집계구. 정량 변화가 미미할 수 있어 위성·이미지 기반 시각화 병행
    • 가로수 레이어는 공공데이터포털 가로수 데이터(과거 2013년분 포함) 활용 검토, 토지피복도 산림 마스킹 전처리(A) vs 기존 방식(B) 동대문구 A/B 테스트 후 채택

    ▶ 다음 업무

    • LocateAnything 7개 지역(서울·인천·부산·광주·대전·울산·대구) 모델링 완료(약 80%) 즉시 결과 공유·공동 육안 검토 (~6/25 예상)
    • 동대문구 SAM 정답지 제작 — 환경이 다른 2개 지역 각 15~20장(총 30~40장) 무작위 추출
    • 기온 LST 격자 점수화·300m 공원 DB 구축(도시공원 포털+브이월드)·동대문구 3m/30m 분석 (6/30 목표)
    • 제기동 식재 전·후 시나리오 수치 산정 및 위성 기반 시각화 준비
  4. 중간 점검 회의 (착수 3개월) 2026.06.15

    • 대상지 범위 확장 — 선정 7개 자치구 내 모든 행정동에 K-TES 산출, '자치구 내 행정동 비교'를 기본 분석 단위로 확정
    • 수목 탐지 최종 모델로 LocateAnything v8 채택 — 이상치는 코드 추출 후 사람이 검토·수정하는 2단계로 처리
    • 정확도 평가는 (사람+SAM) vs (모델+SAM) 일치율 비교 방식 채택(용역용), 논문용 엄격 검증은 후속 과제로

  5. 구도심 녹지화 진행 점검 회의 2026.05.28

    식재 가이드라인 15종을 분석해 식재 가능 공간을 가로·유휴부지·공공건축물 3유형으로 분류하고 '공간유형×관리주체×식재전략' 3단계 유형체계를 정립. 대상지 3곳(서울 제기동·울산 화정동·부산 당리동)에 식재 시뮬레이션을 적용해 예상 식재 그루 수와 추가 수관면적을 산출. 취약지역 분석 단위는 집계구를 목표로 하되 지표 데이터 한계 시 행정동 대체를 검토하고, 산림청 측 가로수 위치 데이터는 미검증 상태로 즉시 활용이 어려움을 확인.

  6. 한국형 3-30-300 지표 기준 확정 회의 2026.05.13

    ✓ 결정 사항

    • 3-30-300 기준 확정 — 층수별 버퍼(저층 30m/중층 55m/고층 100m), '30' 사적녹지 포함, '300' 300m 네트워크
    • 거점도시 7곳 대상지 선정, 분석단위를 행정동 → 집계구로 세분화
    • DeepForest 옥상 초록색 오탐 확인 → 산림청 가로수 데이터로 파인튜닝하기로

    ▶ 다음 업무

    • 산림청 가로수 데이터 수령(외장하드) 후 수관탐지 모델 파인튜닝 착수
    • 데이터 도착 전 거점도시 1~2곳 K-TES 점수 시뮬레이션 선행
  7. 산림청 착수보고 2026.04.29

    K-TES 설계방향(글로벌 스탠다드 수용 + 한국 고밀도·기후취약성 가점) 보고. Tier2 대상 4개 광역시(서울·인천·대전·부산) 확정, 최종납품 2026-11-30 확정.

  8. 착수보고 준비 회의 2026.04.20

    층수별 가시권 차등, NDVI 임계값 0.5 적용성, 배후산림·사적녹지 포함 여부 논의. 발주처 기준 부재 → 연구팀 기준을 착수보고에서 직접 제안하기로.

  9. 내부 방법론 1차 회의 2026.04.17

    3-30-300 한국형 측정법 검토. '3'=위성+건물높이, '30'=항공영상 해상도 한계로 Tier1/2 이원 전략, '300'=도로 네트워크 최단경로 채택.

과제 개요

WHO/UNECE 수관피복 권고와 3-30-300 규칙을 한국 맥락으로 재해석하여, 기존의 면적 중심 도시숲 정책을 형평성 기반의 단계적 조성 전략으로 전환하는 연구다. 기후·사회·경제 취약성 지표를 통합한 K-TES(한국형 Tree Equity Score) 를 설계하고, 2SFCA 접근성·NDVI 수관피복률을 결합해 Tier 1(전국·공공데이터) 과 Tier 2(거점도시·고해상도) 이원 분석체계로 우선 조성 대상지를 도출한다.

  • 분석 방법: 3-30-300, K-TES(GAPScore × KEI), G2SFCA/MAH-G2SFCA, GVI(녹시율), NDVI 기반 수관피복(TCC), ODI(구도심 지수), A/B/C Zone 분류
  • 과업 4단계: 가. 실태조사 → 나. 지표개발 → 다. 전략마련 → 라. 기타(심포지엄·홍보)
  • 수행기관: 서울시립대학교 산학협력단 / SDC Lab
