과제 개요

Arnstein의 참여 사다리와 IAP2 스펙트럼을 이론 기반으로, 4대 생성형 AI 도구를 개발·실증하여 형식적(tokenistic) 주민참여를 placation·partnership 수준으로 끌어올린다. 통합 프레임워크는 IRAPM(Inclusive Rural AI Participation Methodology). 3개년(2026.06~2029.05) 과제로, 시범지는 여주 금사면이며 경기도 농촌공간계획 주민참여 실행 용역(양주 은현면)과 보조 트랙으로 연계한다.

4대 방법론

  1. STT+LLM 의견수렴·시각화 (Whisper STT + LLM 분류 + 지도 시각화)
  2. LLM 페르소나·시나리오 (사회과학적 디지털 트윈)
  3. Text-to-Image 미래상 시뮬레이션 (Before/After + 미래뉴스)
  4. RAG 기반 주민 주도 계획지원

연차 구조: 1차(이론+①③ 개발+파일럿 1개 마을) → 2차(②④ 개발+금사면 3개 마을) → 3차(IRAPM 통합+타 시군 확산·정책화)

회의 내용

  • 2026-06-08 · 양주용역 연계방안 확정 — 완전 통합(v1) 대신 부분 겹침(v2) 로 재정렬. 용역의 현황·녹취 데이터를 NRF 베이스라인으로 활용, NRF는 방법론 ③(미래 청사진)을 핵심으로. 협력 지원 계획서 및 AI 산출물 4종 데모 작성.
  • 2026-06-10 · 로캐츠 회의 — 양주 은현면 리빙랩 8회차 워크숍에 4대 AI 접목. 디지털 페르소나의 주민 수용성, 페르소나-주민 직접 채팅 가능성, 취약계층(청년·외국인) 의견수렴 전략 논의. 14회차 데이터 축적 → 56회차 AI 결과 공개·비교 방식 합의.
  • 2026-06-10 · 3자 공식 협의(시립대·로캐츠·경기농수산진흥원) — 4단계 방법론 워크숍 적용안, 문산마을 페르소나 8인 토론 시연, 백사면 녹취록 활용 검증. 1~4회차 녹취·인터뷰를 시립대에 제공, 8회차 미래상 AI 영상 제공. 6/17 착수보고에 NRF 협력 포함.

현재 진행상황

진척도 약 18% — 1차년도 착수 초기(D+약 2주). 계획·선행연구·프로토타입 mock은 완료, 실증 핵심 단계는 미착수.

  • ✅ 3개년 마스터플랜 수립(WBS·임계경로·리스크 관리).
  • ✅ 선행연구 딥리서치 89건 — 4개 영역(AI-시민참여·방법론·고령자농촌·이론) 분석.
  • ✅ 4대 방법론 개념설계 완료.
  • ✅ 프로토타입 ①③ 코드 + mock 테스트 완료 — 의견지도 HTML(샘플 16건, 저신뢰 5건 human review 분기), 폐교 Before/After 이미지 4종 + 미래뉴스 2건 생성.
  • ✅ 양주 용역 연계방안(v2) 확정 + AI 4종 데모.
  • ✅ 제안서 인용 정합성 내부 점검 완료(교정 목록 확보 — 세부 내용은 비공개 처리).
  • ⏳ 착수보고(6/17), IRB 신청, 마을 섭외, 실 API 연동은 착수 전.

앞으로 할 일

  • 즉시(6~7월): 착수보고(6/17), 취약계층 접근계획(6/21), 워크숍 일정·설문 설계(6/24), IRB 신청(임계경로), 여주시청 협의 → 금사면 파일럿 마을 확정, 개발환경·실 API 구축, 노년·방언 STT WER 벤치마크.
  • 워크숍(7~9월): 은현면 1~8회차 — 5회차 의견분류·지도화, 8회차 미래상 AI 영상.
  • 1차년도 후반(2026.09~2027.05): IRB 승인(게이트1)+마을 섭외, 방법론 ①③ v1.0(실데이터 연동·WER 개선), ②④ 개념·UX 설계, 여주 금사면 파일럿(고령자 사용성 N=20), KCI 1편 투고·국내 발표.
  • 2·3차년도 목표: SSCI 1+KCI 2(2차), SSCI 1+KCI 1~2(3차), 정책 가이드라인, IRAPM 매뉴얼, 최종 플랫폼.
