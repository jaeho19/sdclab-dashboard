연구 질문 (RQ)

  • 페르소나 차별성
  • "공공 데이터 기반으로 설계된 LLM 디지털 페르소나는 농촌공간재구조화 정책 시나리오에 대해 이해관계자 유형별로 구별 가능한(distinctive) 수용성 반응을 생성하는가?"
  • 디지털 페르소나의 유효성
  • "LLM 디지털 페르소나가 생성한 정책 수용성 반응은 실제 농촌 주민의 반응과 어느 수준에서 수렴(converge)하고, 어느 지점에서 발산(diverge)하는가?"
  • 사전 탐색 도구로서의 유효성 범위
  • "이중검증(전문가 패널 + 주민 비교)을 통해 확인된 수렴·발산 패턴에 기초할 때, LLM 디지털 페르소나는 참여형 농촌계획의 사전 탐색 도구(pre-exploration tool)로서 어떤 조건에서 유효하며, 어떤 한계를 갖는가?"

기존연구 공백

①농촌 맥락 부재: LLM 페르소나 연구는 주로 도시·정치적 맥락에서 수행되었으며, 농촌공간계획 분야에의 적용은 전무하다. ②한국어 페르소나 검증 부재: 대부분의 Silicon Sampling 연구가 영어권에서 수행되었으며, 한국 농촌 주민 맥락의 LLM 페르소나 유효성 검증이 이루어지지 않았다. ③하이브리드 검증 프레임워크 부재: LLM 출력을 실제 주민 응답과 체계적으로 비교·검증하는 표준 프레임이 확립되지 않았다. ④불일치 유형 분류 부재: LLM 페르소나가 "언제, 왜 실패하는가"에 대한 체계적 유형화가 이루어지지 않았다. ⑤정책 수용성 사전 탐색 도구로서의 실증: LLM 페르소나의 사전 탐색(pre-exploration) 도구로서의 유효성 범위가 실증적으로 제시되지 않았다.
